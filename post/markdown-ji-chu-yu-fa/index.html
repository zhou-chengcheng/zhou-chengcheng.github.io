MarkDown基础语法

· 2019-05-05 · # Markdown

✍️markdown通过简单的语法定义，极为方便的进行文字排版 !
✍️markdown语法其实很简单有效，下面整理的这些就是比较常用到的。

1.斜体和粗体

使用使用 * 来表示斜体，使用 ** 来表示粗体

示例：
这是斜体：斜体 , 这是粗体：粗体

2.标题

使用 #加空格表示一级效果，同理，连续的#表示对应的多个层级效果

3.外链超链接

使用 【描述】（链接地址）为文字增加超链接

示例： 点击这里关注我

4.无序列表

使用 -来表示无序列表 示例：

  • 无序列表1
  • 无序列表2
  • 无序列表3

5.有序列表

使用数字和点表示有序列表 示例：

  1. 有序列表1
  2. 有序列表2
  3. 有序列表3

6.文字引用

使用 > 表示引用

示例：

野火烧不尽，春风吹又生！
